# 🛕 SRPS Temple App

Welcome to the official SRPS Temple Application 🙏

## 📱 Features

- 🟠 Laddu Online Booking System
- 🟢 User Login & Admin Login
- 🔵 Ticket Generation & Print Option
- 🟣 Seva Registration
- 🟡 Devotional UI Design
- Ticket printing for festival purpose only

## 🎯 Purpose

This app is designed to help devotees easily:
- Book laddus online
- Register for seva
- Manage tickets digitally

## 🛠️ Built With

- Kodular
- TinyDB
- Firebase (optional)
